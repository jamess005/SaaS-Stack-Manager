# Business Rules — Project Management Category

## Must-Have Features
- Task and project management with board and list views
- Native time-tracking (to eliminate ChronoLog £95/mo — consolidation opportunity)
- Client portal with commenting and deliverable approval
- Resource capacity planning / utilisation tracking
- Gantt / timeline with auto-updating dependencies
- Native Slack or TeamSync integration (no Zapier workaround)
- Template library for consulting project types

## Nice-to-Have Features
- AI task estimation and scheduling
- Budget vs actuals tracking per project
- Automated status reporting (weekly digest to stakeholders)
- Workload balancing assistant
- Mobile app with offline mode

## Needs / Pain Points (active)
- ChronoLog overlap costs £95/mo and creates a separate data silo — consolidation is a priority
- Client portal gap means deliverable approvals happen over email — version control issues
- Gantt manual refresh causing project delays going unnoticed
- 52 inactive TaskBridge seats at full price — shelfware flagged

## Compliance
- Project data may include client-confidential materials — SOC2 required
- GDPR for client contact data in portal

## Pricing Sensitivity
- Current effective cost: £290 TaskBridge + £95 ChronoLog = £385/mo combined
- A tool that consolidates both should be evaluated against £385/mo, not £290/mo
- Seat count optimisation: if a new tool is priced per active user, 98 active vs 150 total is relevant

## Migration Notes
- Active project data and templates must migrate
- Historical project archive: 2 years minimum
- ChronoLog time-log data: must migrate into new tool if it includes time-tracking
